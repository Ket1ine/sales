# Информационная система для реализации готовой продукции компанией (Вариант 3)

Курсовой проект по дисциплине «Проектирование и администрирование баз данных». Приложение представляет собой клиент-серверную систему автоматизации оптово-розничных продаж готовой продукции компании, реализованную на архитектурном шаблоне MVC с использованием C++ / Qt 6 и СУБД PostgreSQL.

## Требования для запуска
* **СУБД:** PostgreSQL версии 15, 16, 17 или 18 (должен быть запущен локально на порту 5432).
* **Клиент:** Windows 10 / 11 (для запуска готовой сборки) или macOS/Linux (для сборки из исходников).

---

## Инструкция по развертыванию и запуску

### Шаг 1. Настройка базы данных в PostgreSQL
Приложение по умолчанию настроено на подключение к локальной базе данных `product_sales` под системным пользователем `postgres` без пароля.

1. Откройте консоль PostgreSQL (`psql`) или любой GUI-клиент (`pgAdmin`, `DBeaver`) под учетной записью суперпользователя (`postgres`).
2. Создайте базу данных `product_sales`:
   ```sql
   CREATE DATABASE product_sales;
   ```
3. Импортируйте структуру таблиц, начальные данные и связи из файла схемы. В консоли выполните:
   ```bash
   psql -U postgres -d product_sales -f sql/init.sql
   ```
   *(Либо скопируйте содержимое файла `sql/init.sql`, откройте его в SQL-редакторе вашего GUI-клиента и выполните весь скрипт).*

### Шаг 2. Запуск клиентского приложения

#### Вариант А: Запуск готового релиза (Рекомендуется для Windows)
1. Перейдите в папку скомпилированного релиза: `build/Desktop_Qt_6_11_1_MinGW_64_bit_Debug/debug/` (или скачайте соответствующий артефакт из вкладки Releases).
2. Запустите исполняемый файл `product_sales.exe`. 
3. Приложение автоматически подключится к вашей локальной базе данных на порту 5432.

#### Вариант Б: Сборка из исходного кода (Для разработчиков)
1. Установите **Qt SDK 6.x** и среду разработки **Qt Creator**.
2. Убедитесь, что в среде настроен драйвер `QPSQL` (поставляется вместе с Qt для работы с PostgreSQL).
3. Откройте в Qt Creator файл проекта: `product_sales.pro`.
4. Настройте комплект сборки (Kit) под вашу систему (например, *Desktop Qt 6.x.x MinGW 64-bit*).
5. Запустите сборку и выполнение проекта (сочетание клавиш `Ctrl+R` или кнопка с зеленым треугольником **Run**).

## Тестовые учетные записи для входа

В базе данных по умолчанию предзаполнен пользователь с правами Администратора:

* **Администратор** (полный доступ к CRUD товаров, покупателей, скидок, просмотру/удалению сделок и управлению ролями сотрудников):
  * **Логин:** `admin`
  * **Пароль:** `Admin123!`

* **Пользователь / Менеджер** (доступен просмотр товаров, добавление покупателей, оформление документов розничных и оптовых продаж, управление справочниками цен и скидок заблокировано):
  * Учетную запись менеджера можно зарегистрировать самостоятельно прямо на стартовом экране авторизации (кнопка **Регистрация**), либо ее может добавить Администратор через вкладку управления пользователями. При регистрации пароль должен удовлетворять требованиям сложности (не менее 8 символов, минимум одна цифра, одна заглавная буква и один спецсимвол).
